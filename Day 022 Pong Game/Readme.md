# 🏓 Pong Game – Day 22 (100 Days of Code) 👇

A classic Pong arcade game built using Python's built-in `turtle` module as part of Day 22 of the course  
"100 Days of Code: The Complete Python Pro Bootcamp for 2023" by Dr. Angela Yu.

---

## 🚀 Project Overview

This project recreates the classic 2-player Pong game.

### 🎮 Features:
- Two player paddle control
- Ball movement with bounce physics
- Collision detection (paddle & wall)
- Scoreboard system
- Real-time screen update using `tracer()`
- Smooth animation using `time.sleep()`

---

## 🛠️ Built With

- Python 3
- Turtle Graphics Module
- Object-Oriented Programming (OOP)

---

## 🎮 Controls

| Player | Up | Down |
|--------|----|------|
| Right Paddle | ↑ (Up Arrow) | ↓ (Down Arrow) |
| Left Paddle | W | S |
