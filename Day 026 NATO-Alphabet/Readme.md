# Day 26 - NATO Alphabet Project

## Project Description
This project converts any user-input word into the NATO Phonetic Alphabet using Python.

The program reads a CSV file containing NATO alphabet codes and creates a dictionary using dictionary comprehension.  
When the user enters a word, the program converts each letter into its corresponding NATO phonetic code.

Example:
Input: CAT  
Output: ['Charlie', 'Alfa', 'Tango']

---

## Features
- Reads data from a CSV file using Pandas
- Converts CSV data into a Python dictionary
- Uses Dictionary Comprehension
- Takes user input
- Converts each letter into the NATO phonetic equivalent
