# 🐢 Turtle Crossing Game – Day 23 Capstone Project 👇

A Frogger-style arcade game built using Python's `turtle` module as part of  
Day 23 of the "100 Days of Code: The Complete Python Pro Bootcamp for 2023" course by Dr. Angela Yu.

---

## 🚀 Project Overview

In this game, the player controls a turtle that must cross the road while avoiding moving cars.

Each time the turtle successfully reaches the top of the screen:
- The level increases
- The cars move faster
- The difficulty increases

This project focuses heavily on Object-Oriented Programming and game logic structure.

---

## 🎮 Game Features

- Player-controlled turtle movement
- Randomly generated cars
- Increasing difficulty with each level
- Collision detection system
- Scoreboard with level tracking
- Game over detection

---

## 🛠️ Built With

- Python 3
- Turtle Graphics Module
- Object-Oriented Programming (OOP)
- Game loop logic
- Random module
