# India States Guessing Game

This project is a simple interactive game built with Python. The goal of the game is to guess the names of Indian states. When the user correctly guesses a state, the state name appears on the map at the correct location.

This project helped practice working with Python, data files, and graphical interfaces.

## Features

- Interactive map of India using Turtle graphics
- User can guess Indian state names
- Correct guesses appear on the map
- Tracks how many states are guessed correctly
- Uses CSV data for state names and coordinates


## How It Works

1. The program displays a blank map of India using the Turtle graphics module.
2. The user is asked to enter the name of an Indian state.
3. The program checks if the state exists in the dataset.
4. If the guess is correct, the state name appears at the correct coordinates on the map.
5. The game continues until the player guesses all states or exits the game.


## Concepts Used

- Python Turtle graphics
- Pandas library
- CSV data handling
- Loops
- Condition checking
- Basic game logic


This project was built while learning Python and practicing data handling and simple game development.
