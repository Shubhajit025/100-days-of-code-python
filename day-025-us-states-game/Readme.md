# States Guessing Game (Day 25)

This project is part of the 100 Days of Code: Python Bootcamp by Dr. Angela Yu.

The goal of this project is to build an interactive game where the player tries to guess the names of states. When the player correctly guesses a state, its name appears on the map at the correct location.

## Features

- Interactive map using the Turtle graphics module
- User can guess state names
- Correct guesses appear on the map
- Tracks the number of correct answers
- Shows remaining states to guess
- Uses CSV data for state names and coordinates


## How It Works

1. The game displays a map using Turtle graphics.
2. The user enters the name of a state.
3. The program checks if the state exists in the dataset.
4. If the guess is correct, the state name appears on the map.
5. The game continues until all states are guessed or the user exits.


## Concepts Used

- Python Turtle graphics
- Pandas library
- CSV data handling
- Loops and condition checking
- Data filtering with pandas


This project helped practice working with data and building simple interactive games using Python.
