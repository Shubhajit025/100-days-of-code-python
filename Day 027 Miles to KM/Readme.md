# Miles to Kilometers Converter (Day 27)

A simple Python project that converts distance from **miles to kilometers** using user input.

This project was built as part of my **Python learning journey (Day 27)**.

---

## Features

- Takes **distance in miles** as input
- Converts miles to kilometers
- Displays the result

---

## How It Works

The program uses the following formula:

Kilometers = Miles × 1.60934

Steps:

1. The user enters a distance in miles.
2. The program multiplies the value by **1.60934**.
3. The result is printed in kilometers as rounded value to 2 decimal no (Ex. 16.0945 ⩳ 16.09).

Example:

```
Enter miles: 10
Output: 16.09 km
```
