# 🐍 Snake Game (Python Turtle) 👇

A classic Snake Game built using **Python** and the **Turtle graphics module**.

This project was completed as part of Day 20 & Day 21 of my Python learning journey.

---

## 🚀 Features

- Smooth snake movement
- Keyboard controls (Up, Down, Left, Right)
- Random food spawning
- Score tracking system
- Wall & tail collision detection
- Tail growth after eating food
- Game over after collision with wall or tail

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
- Classes & Objects
- Multiple file structure
- Game loop logic
- Collision detection
- Event listeners
- Screen update control
