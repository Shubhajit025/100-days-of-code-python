# Pomodoro Timer App (Day 28)

A simple Pomodoro Timer application built with Python and Tkinter.  
This app helps users stay focused using the Pomodoro productivity technique by alternating between work sessions and breaks.

## Features
- 25 minute Work session
- 5 minute Short break
- 20 minute Long break after multiple work sessions
- Start and Reset buttons
- Automatic timer countdown
- Checkmarks `✓` to track completed work sessions
- Simple and clean UI built with Tkinter

## Technologies Used
- Python
- Tkinter (GUI)
- Math module

## How It Works
1. Click **Start** to begin the timer.
2. The timer will start a work session.
3. After the work session ends, a short break starts automatically.
4. After several sessions, a long break is triggered.
5. Each completed work session adds a `✓` checkmark to track productivity.
